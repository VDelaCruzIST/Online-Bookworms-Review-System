# Online Bookworms Review System

## Overview
This repository contains an online book review website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website allows users to rate and review books through an interactive review system. The project demonstrates JavaScript event handling, dynamic user interaction, character counting, rating systems, DOM manipulation, and responsive web design concepts.

## Features
* Interactive book review platform
* Star-based rating system
* Dynamic rating selection
* Real-time character counting
* User review submission interface
* Input validation feedback
* Book information and review display
* Responsive page layout

## Technologies Used
* HTML5
* CSS3
* JavaScript
* DOM Manipulation
* Event Handling
* Regular Expressions

## Project Files
* `bw_review_txt.html` – Main review webpage
* `bw_review_txt.js` – Interactive review and rating functionality
* `bw_base.css` – Base stylesheet
* `bw_layout.css` – Layout and visual styling
* Book cover, icon, and branding assets

## Skills Demonstrated
* Front-end web development
* JavaScript programming
* Event-driven programming
* DOM manipulation
* Real-time user input processing
* Character counting and validation
* Interactive user interface design
* User experience development

## Academic Project
This project was completed as part of web development coursework and demonstrates JavaScript programming concepts used in interactive web applications, including event handling, dynamic content updates, user input validation, and rating system implementation.
